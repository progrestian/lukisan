# Lukisan

Artwork fetcher and wallpaper manager

## Support

### Fetch sources

- DeviantArt

### Manager backends

- SwayWM

## Usage

```
// clone the repo
git clone https://github.com/progrestian/lukisan.git

// install all dependencies
npm install

// run the npm script to compile
npm run build

// launch the compiled dist/index.js
node dist/index.js
```
